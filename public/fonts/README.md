# Bundled subtitle fonts

Vector (enhanced-mode) subtitle typefaces. All are open-source and bundled so the
subtitles render identically on every platform. Each is a variable font (weight
axis 100–900); the game uses the Medium (500) weight.

| Font | License | Upstream |
|------|---------|----------|
| Mulish (default) | SIL OFL 1.1 — see `OFL-Mulish.txt` | https://github.com/googlefonts/mulish |
| Manrope | SIL OFL 1.1 — see `OFL-Manrope.txt` | https://github.com/sharanda/manrope |
| Jost | SIL OFL 1.1 — see `OFL-Jost.txt` | https://github.com/indestructible-type/Jost |

The original FreeSans Bold subtitle face lives at `../enhanced/subtitle.ttf`
(GNU GPL v3 with the font exception; see `../enhanced/subtitle-FONT-LICENSE.txt`).

Mulish Medium was chosen as the default as a clean, open humanist face close in
look to Avenir Next Medium. The in-game font previewer (F key, enhanced mode)
cycles the bundled alternates.
